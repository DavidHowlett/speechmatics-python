speechmatics-python
===================

speechmatics-python provides the Python API and command-line interface for Speechmatics.


Getting started
---------------

speechmatics-python can be installed via `PyPI <https://pypi.org/project/speechmatics>`_. Official support is available for Python 3.6 and above.
